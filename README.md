# Mon Portfolio Web

## Présentation
Ce portfolio présente mes projets en développement web. Il contient des exemples de sites que j'ai réalisés, mes compétences techniques et mes coordonnées.

## Contenu
- **Page principale :** Une présentation de mes compétences et mes projets.
- **Projets :** Chaque projet est décrit avec des images et des explications.

## Comment l'utiliser ?
1. Téléchargez le dossier du portfolio.
2. Ouvrez le fichier `index.html` dans un navigateur web (Google Chrome, Firefox, etc.).

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- PHP
